# API_ALURALATAM

Este repositorio contiene una API REST desarrollada en Java con Spring Boot, realizada como parte de una práctica de aprendizaje en las clases de Alura Latam.

El objetivo principal del proyecto es reforzar conceptos fundamentales del desarrollo backend, como la creación de endpoints, el uso de Maven, y la integración con bases de datos SQL.

## Sobre este proyecto

Este proyecto fue desarrollado durante el programa de formación de Alura Latam (Oracle Next Education - ONE) como ejercicio práctico en clase.

Es una API creada con fines educativos para comprender:

- Cómo estructurar un proyecto backend en Java
- Cómo construir una API REST con Spring Boot
- Cómo conectar el backend con una base de datos SQL
- Cómo utilizar Git y GitHub para versionar proyectos

## Tecnologías utilizadas

- Java
- Spring Boot
- Maven
- SQL (Base de datos relacional)
- IntelliJ IDEA (entorno de desarrollo)
- Git y GitHub

## Aprendizajes principales

Con esta práctica pude trabajar en:

- Creación de controladores REST
- Definición de rutas y endpoints
- Uso de Spring Boot para levantar un servidor backend
- Integración con bases de datos SQL
- Buenas prácticas de organización en proyectos Java
- Publicación de proyectos en GitHub

## Cómo ejecutar el proyecto

1. Clonar el repositorio desde GitHub
2. Entrar a la carpeta del proyecto
3. Ejecutar con Maven usando el comando mvn spring-boot:run

## Nota

Este proyecto es una práctica académica realizada durante las clases de Alura Latam, por lo tanto está enfocado en el aprendizaje y no en un desarrollo productivo final.
