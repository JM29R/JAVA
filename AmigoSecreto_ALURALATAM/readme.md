# AmigoSecreto_ALURALATAM

Este repositorio contiene un proyecto sencillo llamado **Amigo Secreto**, desarrollado como parte de una práctica de clase en Alura Latam. Es un ejercicio básico para practicar lógica de programación y uso de Java.

## Sobre este proyecto

Es una práctica realizada durante el curso de Alura Latam para comprender conceptos fundamentales de programación en Java y practicar estructuras básicas, lógica, y organización de código.

## Tecnologías utilizadas

- Java
- Git & GitHub

## Qué se aprendió

Con este proyecto se trabajó en:

- Lógica de programación en Java
- Organización de clases y métodos
- Uso de Git y GitHub para subir y versionar código

## Cómo ejecutar

1. Clonar el repositorio desde GitHub
2. Abrir el proyecto en un entorno de desarrollo (como IntelliJ IDEA)
3. Compilar y ejecutar con Java

## Nota

Este proyecto es una práctica básica realizada en clases de Alura Latam y no está diseñado como un proyecto productivo.

Gracias por visitar este repositorio.
